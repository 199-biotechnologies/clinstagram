# Clinstagram Audit & Optimization Mission

## Goal
Comprehensive audit of the clinstagram project to identify performance optimizations, security hardening, latency reduction, competitive gaps, and feature opportunities.

## Scope
- Full codebase audit (src/clinstagram/, tests/)
- Performance & latency analysis
- Security & anti-detection hardening
- Competitive analysis (instagram-cli, instagrapi ecosystem)
- CLI skill installation capability research
- instagrapi upstream changes & best practices

## Success Criteria
- Actionable improvement recommendations ranked by impact
- Performance bottleneck identification with specific fixes
- Security hardening checklist
- Competitive feature gap analysis
- Skill installation design for Claude/Gemini
